## Description

Demo project for Apache Kafka

## Getting Started

### Prerequisites:
*   Java 21 or higher
*   Maven 3.x or higher

### Installation:
*   Clone the repository: `git clone https://github.com/example/driver-sdk.git`
*   Build the project: `mvn clean install`

## Running Tests

Run tests using Maven: `mvn test`
